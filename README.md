# 📈 Real-Time Chart Application

A real-time chart application built with **React**, **TypeScript**, and **Redux** that displays live data updates with a fast and responsive user interface.

---

## ✨ Features

* Real-time data visualization
* Live chart updates
* State management with Redux
* Responsive modern UI
* Reusable and scalable components

---

## 🛠️ Tech Stack

* React
* TypeScript
* Redux
* WebSocket

---

## 🚀 Run Locally

npm install
npm run start
```

src/
```
